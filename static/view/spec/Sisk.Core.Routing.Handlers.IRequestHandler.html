IRequestHandler interface

Definition: 


public interface IRequestHandler

Represents an interface that is executed before a request.

Method list

Execute(HttpRequest, HttpContext) This method is called by the before executing a request when the instantiates an object that implements this interface. If it returns a object, the route callback is not called and all execution of the route is stopped. If it returns "null", the execution is continued.

Property list

ExecutionMode Gets or sets when this RequestHandler should run.
Identifier Gets or sets the unique identifier of the instance of this interface.
