Execute(HttpRequest, HttpContext) method

Declaring type:

Sisk.Core.Routing.Handlers.IRequestHandler

Definition: 

HttpResponse? Execute(HttpRequest request, HttpContext context);

This method is called by the before executing a request when the instantiates an object that implements this interface. If it returns a object, the route callback is not called and all execution of the route is stopped. If it returns "null", the execution is continued.
