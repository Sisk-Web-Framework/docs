HttpServerFlags type

Assembly: Sisk.Core

Provides advanced fields for Sisk server behavior.

Field list

HeaderNameRequestId Determines the HTTP header name of the request ID.
OptionsLogMode Determines if the HTTP server should write log to OPTIONS requests.
SendCorsHeaders Determines if the HTTP server automatically should send CORS headers if set.
SendSiskHeader Determines if the HTTP server should send the X-Powered-By header in all responses.
ThrowContentOnNonSemanticMethods Determines if the HTTP server should drop requests which has content body in GET, OPTIONS, HEAD and TRACE methods.
TreatHeadAsGetMethod Determines if the HTTP server should automatically send HTTP headers of an pre-processed GET response if the request is using HEAD method.
