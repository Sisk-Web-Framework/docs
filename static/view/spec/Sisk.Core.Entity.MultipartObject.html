MultipartObject class

Definition: 


public class MultipartObject

Represents an multipart/form-data object.

Property list

ContentBytes The multipart form data content bytes.
ContentLength The multipart form data content length.
DefaultContentEncoding Gets or sets the default content encoding for decoding objects contents as strings.
DefaultHeadersEncoding Gets or sets the default content encoding for decoding multipart-form headers names and values.
Filename The name of the file provided by Multipart form data. Null is returned if the object is not a file.
Headers The multipart form data object headers.
Name The multipart form data object field name.

Method list

GetCommonFileFormat() Determine the image format based in the file header for each image content type.
ReadContentAsString() Reads the content bytes as an ASCII string.
ReadContentAsString(Encoding) Reads the content bytes with the given encoder.
