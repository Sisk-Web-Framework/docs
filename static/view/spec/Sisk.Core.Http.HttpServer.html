HttpServer class

Assembly: Sisk.Core

Definition: 

public class HttpServer : IDisposable

Provides an lightweight HTTP server powered by Sisk.

Property list

EventSources Gets an HttpEventSourceCollection with active event source connections in this HTTP server.
IsListening Gets an boolean indicating if this HTTP server is running and listening.
ServerConfiguration Gets or sets the Server Configuration object.

Event list

OnConnectionClose Event that is called when this HttpServer computes an request and it's response.
OnConnectionOpen Event that is called when this HttpServer receives an request.

Method list

Dispose() Invalidates this class and releases the resources used by it, and permanently closes the HTTP server.
GetVersion() Get Sisk version label.
Restart() Restarts this HTTP server, sending all processing responses and starting them again, reading the listening ports again.
Start() Starts listening to the set port and handling requests on this server.
Stop() Stops the server from listening and stops the request handler.

Constructor list

HttpServer(HttpServerConfiguration) Creates a new default configuration HttpServer instance with the given Route and server configuration.
