KeepAlive(TimeSpan) method

Declaring type: Sisk.Core.Http.Streams.HttpRequestEventSource (from Sisk.Core)

Definition: 

public void KeepAlive()

Asynchronously waits for the connection to close before continuing execution with an maximum keep alive timeout. This method is released when either the client or the server reaches an sending failure.

Parameters

maximumIdleTolerance The maximum timeout interval for an idle connection to automatically release this method.
