LogOutput enum

Definition: 


public enum LogOutput

Determines the way the server can write log messages. This enumerator is for giving permissions for certain contexts to be able or not to write to the logs.

Field list

AccessLog Determines that the context or the route can write log messages only to the access logs.
Both Determines that the context or the route can write log messages to both error and access logs.
ErrorLog Determines that the context or the route can write error messages only to the error logs.
None Determines that the context or the route cannot write any log messages.
