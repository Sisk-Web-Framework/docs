public enum RequestHandlerExecutionMode

Defines when the IRequestHandler object should be executed.

Field list

AfterResponse Indicates that the handler must be executed after the route callback execution.
BeforeContents Indicates that the handler must be executed before the router calls the route callback and before the request content is available.
BeforeResponse Indicates that the handler must be executed before the router calls the route callback and after request contents is loaded.
