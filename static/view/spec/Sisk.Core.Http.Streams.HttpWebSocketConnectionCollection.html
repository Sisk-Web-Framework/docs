Assembly: Sisk.Core

Definition: 

public class HttpWebSocketConnectionCollection

Provides a managed object to manage HttpWebSocket connections.

Event list

OnWebSocketRegister Represents an event that is fired when an HttpWebSocket is registered in this collection.
OnWebSocketUnregister Represents an event that is fired when an HttpWebSocket is closed and removed from this collection.

Method list

All() Gets all actives HttpWebSocket instances.
DropAll() Closes all registered and active HttpWebSocket in this collections.
Find(string, Boolean}) Gets all actives HttpWebSocket instances that matches their identifier predicate.
GetByIdentifier(string) Gets the Web Sockect connection for the specified identifier.
