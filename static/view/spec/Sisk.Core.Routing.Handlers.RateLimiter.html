RateLimiter class

Definition: 

public class RateLimiter : IRequestHandler

Represents a rate limiter that is executed after the socket receives the HTTP message from the server.

Property list

ExecutionMode Gets or sets when this RequestHandler should run.
Identifier Gets a unique identifier for this instance.
LimitingPolicy Gets or sets the routing limitation policy settings for this RateLimiter.
Repository Gets or sets the cache of requests that this Rate Limiter stores.

Constructor list

RateLimiter(RateLimiterRepository, RateLimiterPolicy) Creates a new instance with given parameters.

Method list

Execute(HttpRequest, HttpContext) Executes the rate limiter action on the route and checks if it will be blocked or not.
