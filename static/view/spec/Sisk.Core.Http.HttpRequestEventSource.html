HttpRequestEventSource class

Assembly: Sisk.Core

Definition: 

public class HttpRequestEventSource : IDisposable

An HttpRequestEventSource instance opens a persistent connection to the request, which sends events in text/event-stream format.

Property list

Identifier Gets or sets an label to this EventStream connection, useful for finding this connection's reference later.
IsActive Gets an boolean indicating if this connection is open and this instance can send messages.
SentContentLength Gets an integer indicating the total bytes sent by this instance to the client.

Method list

AppendHeader(string, string) Sends an header to the streaming context.
Cancel() Cancels the sending queue from sending pending messages and clears the queue.
Close() Closes the event listener and it's connection.
Dispose() Flushes and releases the used resources of this class instance.
Equals(Object) Determines if another object is equals to this class instance.
GetHashCode() Gets the hash code for this event source.
KeepAlive() Asynchronously waits for the connection to close before continuing execution. This method is released when either the client or the server reaches an sending failure.
KeepAlive(TimeSpan) Asynchronously waits for the connection to close before continuing execution with an maximum keep alive timeout. This method is released when either the client or the server reaches an sending failure.
Send(Object) Writes a event message with their data to the event listener and returns an boolean indicating if the message was delivered to the client.
Send(string) Writes a event message with their data to the event listener and returns an boolean indicating if the message was delivered to the client.
