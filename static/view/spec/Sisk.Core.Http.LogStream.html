LogStream class

Assembly: Sisk.Core

Namespace: Sisk.Core.Http

Definition: 

public class LogStream : IDisposable

Provides a managed, asynchronous log writer which supports writing safe data to log files or streams.

Field list

ConsoleOutput Represents a LogStream that writes its output to the Out stream.

Property list

Encoding Gets or sets the encoding used for writting data to the output file. This property is only appliable if this instance is using an file-based output.
FilePath Gets the absolute path to the file where the log is being written to.
TextWriter Gets the TextWriter object where the log is being written to.

Constructor list

LogStream(TextWriter) Creates an new LogStream instance with the given TextWriter object.
LogStream(string) Creates an new LogStream instance with the given relative or absolute file path.

Method list

Close() Writes all pending logs from the queue and closes all resources used by this object.
Dispose() Writes all pending logs from the queue and closes all resources used by this object.
Peek(int) Reads the last few lines of the linked log file.
Set() Releases the execution of the queue.
Wait(bool) Waits for the log to finish writing the current queue state.
Write(Object) Writes the text into the output.
WriteException(Exception) Writes an exception description in the log.
WriteLine() Writes an line-break at the end of the output.
WriteLine(Object) Writes the text and concats an line-break at the end into the output.
WriteLine(string, Object[]) Writes the text format and arguments and concats an line-break at the end into the output.
