ServiceProvider class

Definition: 


public class ServiceProvider

Provides a class that organizes and facilitates the porting management of a service or application that uses Sisk.

Property list

AccessLogs Gets the configured access log stream. This property is inherited from .
ConfigurationFile Gets or sets the Sisk server portable configuration file.
ErrorLogs Gets the configured error log stream. This property is inherited from .
Flags Gets or sets advanced configuration settings for the HTTP server initialization.
HttpServer Gets the emitted HTTP server object instance interpreted from the configuration file.
Initialized Gets an boolean indicating if the configuration was successfully interpreted and the server is functional.
RouterFactoryInstance Gets or sets the object instance which will provide an entry point for this service.
ServerConfiguration Gets the emitted server configuration object interpreted from the configuration file.
Verbose Gets or sets whether this should write mensagens to console indicating if the server is listening or not.

Constructor list

ServiceProvider(RouterFactory, string) Creates an new instance with given router factory and custom settings file name.
ServiceProvider(RouterFactory) Creates an new instance with given router factory.

Method list

Initialize() Opens and reads the configuration file, parses it and starts the HTTP server with the router and settings parsed from the file.
Initialize(bool) Opens and reads the configuration file, parses it and starts the HTTP server with the router and settings parsed from the file.
