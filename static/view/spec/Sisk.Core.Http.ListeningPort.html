ListeningPort struct

Definition: 

public struct ListeningPort

Provides a structure to contain a listener port for an instance.

Property list

Port Gets or sets the port where the server will listen.
Secure Gets or sets whether the server should listen to this port securely (SSL).

Constructor list

ListeningPort(int, bool) Creates an new instance with the specified port and secure context.
ListeningPort(int) Creates an new instance with the specified port.
