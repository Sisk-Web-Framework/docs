GetEventSource(string) method

Declaring type: Sisk.Core.Http.HttpRequest (from Sisk.Core)

Definition: 

public HttpRequestEventSource GetEventSource(string? identifier = null)

Gets an Event Source interface for this request. Calling this method will put this HttpRequest instance in it's event source listening state.

Parameters

identifier Defines an label to the EventStream connection, useful for finding this connection's reference later.
