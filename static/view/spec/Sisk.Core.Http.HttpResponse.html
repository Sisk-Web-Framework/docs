HttpResponse class

Assembly: Sisk.Core

Namespace: Sisk.Core.Http

Definition: 

public class HttpResponse

Represents an HTTP Response.

Method list

CreateEmptyResponse() Creates an new empty HttpResponse with no status code or contents. This will cause to the HTTP server to close the connection between the server and the client and don't deliver any response.
CreateRedirectResponse(string) Creates an new redirect HttpResponse with given location header.
GetRawHttpResponse(bool) Gets the raw HTTP response message.
SetCookie(string, string, DateTime?, TimeSpan?, string, string, bool?, bool?, string) Sets a cookie and sends it in the response to be set by the client.
SetCookie(string, string) Sets a cookie and sends it in the response to be set by the client.

Property list

Content Gets or sets the HTTP response body contents.
CustomStatus Gets or sets an custom HTTP status code and description for this HTTP response. If this property ins't null, it will overwrite the Status property in this class.
DefaultEncoding Gets or sets the default encoding when creating new HttpResponse instances.
Headers Gets a NameValueCollection instance of the HTTP response headers.
SendChunked Gets or sets whether the HTTP response can be sent chunked.
Status Gets or sets the HTTP response status code.

Constructor list

HttpResponse() Creates an new HttpResponse instance with HTTP OK status code and no content.
HttpResponse(int, HttpContent) Creates an new HttpResponse instance with given status code and HTTP content.
HttpResponse(int) Creates an new HttpResponse instance with given status code.
HttpResponse(HttpContent) Creates an new HttpResponse instance with given HTTP content, with default status code as 200 OK.
HttpResponse(HttpStatusCode, HttpContent) Creates an new HttpResponse instance with given status code and HTTP contents.
HttpResponse(HttpStatusCode) Creates an new HttpResponse instance with given status code.
